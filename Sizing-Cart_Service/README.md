## Requirements

Run the following commands when starting the project:
   
  - npm install -g webpack
  - npm install
  - npm run seed
  - npm run build
  - npm run start

## Development

  - Linter = Eslint with AirBnB style guide
  - Tester = Jest with Enzyme
  - Database = Mondodb with Mongoose
  - Server = Node with Express
